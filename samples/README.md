# Sample artifacts

Curated AI-style deliverables for demos and `npm run seed:more`.

| File | Source tool | Type |
|------|-------------|------|
| `claude-*.html` | Claude | UX mockups, widgets |
| `gamma-*.html` | Gamma | Slide decks |
| `gpt-*.html` | GPT | Docs, runbooks, research |
| `midjourney-*.html` | Midjourney | Visual concepts (HTML stand-ins) |

Add entries to `manifest.json`, drop files in `artifacts/`, then:

```bash
npm run seed:more
npm run index
```
